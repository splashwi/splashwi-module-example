# Splash-Wi Example Module

[![Splash-Wi](https://splash-wi.com/assets/images/poweredby.png)](https://splash-wi.com)

This is an example module for Splash-Wi (v2.0 and up), including an example for how to use this module. Splash-Wi uses a standard folder-based module system. Modules are loaded automatically if you do not link it to a module indicator in the database (see "spl_modules" on standard installation). This module runs on PHP 5.2 and up!
### Installation
Download the files and put the HelloWorld folder into "/app/Modules". Now you can visit "yoururl.com/helloworld" to see the result. For further instructions on how to use the Module-API see: [API reference](https://splash-wi.com/docs)
